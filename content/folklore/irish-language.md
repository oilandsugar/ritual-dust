## Irish alphabet

The traditional Irish alphabet consists of 18 letters <span class="gaeilge">a, b, c, d, e, f, g, h, i, l, m, n, o, p, r, s, t, u</span>. It doesn't include the letters: j, k, q, v, w, x, y, z although these can be used for loanwords.

Vowels can be accented with a <span class="gaeilge">fada</span> like so <span class="gaeilge">á, é, í, ó, ú</span> to indicate a long vowel. 

Consonnants can be accented with a dot <span class="gaeilge">bh, ch, dh, fh, gh, mh, ph, sh, th</span> which means their sound is softened, the equivalent of following the letter with an h in the roman type.
